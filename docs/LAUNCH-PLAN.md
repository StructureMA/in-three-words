# In Three Words — Launch Plan

## April 2026 Timeline

### Week 1 (Apr 6–12): Setup
- Set up Instagram @inthreewords
- Create Google Form for submissions
- Set up Pirate Ship account
- Finalize landing page
- Pick first charity

### Week 2 (Apr 13–19): Launch
- Launch announcement post
- Share with friends/family
- Open for first submission
- Start painting

### Week 3 (Apr 20–26): First Delivery
- Finish first painting
- Photograph, pack, ship
- Post the reveal (buyer's name, 3 words, charity)
- Donate to charity
- Open second submission

### Week 4 (Apr 27+): Evaluate
- What went well? What needs adjusting?
- Refine the process
- Keep the cycle going

## Weekly Content Cycle

1. **Announce the 3 words** (start of week) — post the words visually, simple typography or handwritten on canvas
2. **Work-in-progress tease** (mid-week) — close-up, palette shot, workspace. Don't reveal the whole painting yet
3. **The reveal post** (end of week) — full painting, buyer's name, charity, the 3 words. Tag the charity
4. **Story/reel** — show the painting being packed and shipped

## Platforms & Tools

| Need | Tool |
|------|------|
| Submissions | Google Form — embedded on landing page or linked from Instagram bio |
| Payments | Venmo or PayPal — manual at this scale |
| Shipping | Pirate Ship — discounted USPS rates, no subscription |
| Social Media | Instagram (primary), cross-post to TikTok and Facebook as you grow |
| Landing Page | Static HTML — deployed via GitHub Pages or Vercel |
| Tracking | Simple spreadsheet: buyer name, 3 words, size, payment, ship date, charity |
